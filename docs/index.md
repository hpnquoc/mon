## Installation

```shell
git clone https://phlong3105@github.com/phlong3105/mon
cd mon
chmod +x install.sh
conda init bash
bash -i install.sh
```

Or read the [advanced install guide](get-started/installation.md)

The code is fully compatible with [PyTorch](https://pytorch.org/) version 2.0 
and above.
